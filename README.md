## Installation
1. Clone this repo.
2. `yarn`

## How to run (dev mode)

1. `yarn dev`

Runs the app in the development mode.

Open [http://localhost:3000](http://localhost:3000) to view it in the browser or seee it
on live (https://val089.github.io/music-albums)
